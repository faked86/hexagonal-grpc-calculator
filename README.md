# hexagonal-grpc-calculator
Calculator API that can't process zeroes :(
Uses hexagonal architecture and gRPC protocol.
Have tests in it.

##  Usage
1) Clone repo
2) Run `docker compose up --build`
3) Enjoy!

## Tests
Run `go test -v ./...`
